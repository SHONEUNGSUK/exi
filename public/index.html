🔌 ISO 15118-20:2022

        
전기차(EV)와 충전소(EVSE) 사이의 V2G 통신 표준. AC/DC 충전, 양방향 전력 전송(BPT), 스마트 충전 스케줄을 정의합니다.

      

      

        
⚡ W3C EXI 1.0

        
Efficient XML Interchange — XML을 이진 포맷으로 압축하는 W3C 표준. IoT·자동차 네트워크에서 XML 대비 60~85% 크기를 절감합니다.
